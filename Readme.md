# 📰 Notícias Cidade

O **Notícias Cidade** é um projeto de site de notícias desenvolvido com HTML5 e CSS3, com o objetivo de praticar a construção de layouts estruturados, navegação entre páginas e organização de conteúdo.

---

## 📌 Sobre o Projeto

Este projeto simula um portal de notícias com diferentes categorias, páginas internas e variações de layout (uma, duas e três colunas).

O site possui:
- Página inicial com destaques e notícias recentes
- Páginas específicas como Brasil, Fotos e Nova Legislação
- Sistema de navegação entre páginas
- Layout adaptado para diferentes seções
- Barra lateral com entrevistas e formulário de newsletter

---

## 🧠 Conceitos aplicados

Durante o desenvolvimento, foram trabalhados conceitos como:

- Estruturação semântica com HTML
- Organização de layout utilizando divs e IDs
- Criação de layouts com múltiplas colunas
- Estilização com CSS externo
- Menus de navegação horizontal
- Formulários HTML
- Reutilização de classes e padronização visual

---

## 🎯 Objetivo

O principal objetivo deste projeto foi reforçar a base em desenvolvimento front-end, entendendo melhor como estruturar sites com múltiplas páginas e manter um padrão visual consistente.
